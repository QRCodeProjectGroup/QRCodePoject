# QR MENU
## About This Project

	In this project we aim to create an QR code digital menu that restaurants can use
	during the pandemic process and reduce the close contact.
    

## Built With

* Html/Css/Bootstrap
* React JS
* Node JS
* Express JS
* MongoDB Atlas
* Travis CI
* Heroku
